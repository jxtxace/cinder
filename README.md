# Cinder ($ASH)

A portfolio DeFi project demonstrating modern smart contract architecture with a burn and cashback loop.

## Structure
- **cinder-contracts**: Hardhat environment with Solidity smart contracts, tests, and deployment scripts.
- **cinder-frontend**: React + Vite frontend application utilizing Wagmi and RainbowKit for Web3 interactions.

*Note: This is an unaudited testnet demo and should not be used in production with real funds.*
